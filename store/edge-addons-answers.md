# Microsoft Edge Add-ons — Submission Answers

Canonical answers for the Edge Add-ons Partner Center. Edge uses the same Chromium permission model as Chrome, so most fields reuse the Chrome answers verbatim.

> **Quick reference:** Edge's privacy/permissions questions are identical to Chrome. Copy from [chrome-webstore-answers.md](./chrome-webstore-answers.md) for:
> - Single purpose description
> - Permission justifications (browsingData, tabs, contextMenus, scripting, webRequest, `<all_urls>`)
> - Data usage checkboxes (uncheck all 9)
> - Three certification disclosures (check all 3)
> - Privacy policy URL
>
> Answers below are Edge-specific additions.

## Edge-specific differences from Chrome

- **No "remote code" question** — Edge does not ask about remote code explicitly (Chrome does). Nothing to answer.
- **Age rating required** — Chrome does not require a per-region age rating; Edge does (answer: Everyone / 3+).
- **Search terms are a dedicated field** — Chrome indexes the description; Edge has a 7-term field.
- **Social networks + cookies questions** — Edge-specific. Answer: No / No.
