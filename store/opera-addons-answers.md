## Privacy policy — Full text (paste as fallback)

If Opera requires the full text in the textarea (not just a URL):

```
ClearCache Privacy Policy

The ClearCache browser extension collects no user data. It transmits no
user data. It contacts no remote servers. It stores no user data.

What the extension does:
- Reads the URL and ID of the currently active tab to determine which
  site's cache to clear.
- Calls the browser's built-in browsingData API to clear HTTP cache,
  Cache Storage, Service Workers, and (in deep-clear mode only) cookies,
  localStorage, and IndexedDB for that single site.
- Reloads the active tab with cache bypass.
- Injects a small confirmation toast into the reloaded page for ~2
  seconds, then removes it.
- Observes the tabId and fromCache flag of subresource requests for 4
  seconds after the reload in order to display a "fresh vs cached"
  counter in the toast. These counters are held in memory and discarded
  after the toast closes.

What the extension does NOT do:
- No analytics. No error reporting. No usage tracking.
- No telemetry to any remote endpoint.
- No reading or collecting of page content, cookies, form data,
  passwords, or any personally identifiable information.
- No persistent storage of user data (no localStorage, no IndexedDB,
  no chrome.storage).
- No network requests of any kind initiated by the extension.

The full source code is available at
https://github.com/isaiasgv/ClearCache — the single background.js file
(~350 lines) contains every network-facing and data-access call the
extension makes. Users and reviewers can audit it in full.

Contact: https://github.com/isaiasgv/ClearCache/issues
```

**Length:** ~1,440 chars (fits most textareas; trim the "What does/doesn't do" bullets if Opera has a hard cap).

### English (en)

**Summary** (catalog listing — keep short):

```
Click to clear the cache and hard-reload the current site.
```

**Description** (details page):

```
Ever updated a website but still see the old version? ClearCache fixes that in one click.

Click the toolbar icon and ClearCache wipes the cache for the site you are on and reloads the page, so you always see the latest version. Other sites are never touched.

A small confirmation appears in the top-right of the page so you know it worked, with a quick count showing how many files came back fresh vs. still cached.

Four modes, each with a keyboard shortcut and a right-click menu entry:

• Clear this site and reload — Alt+Shift+R (default click)
• Clear all sites and reload this tab — Alt+Shift+A
• Deep clear this site (cache, cookies, and storage) and reload — Alt+Shift+D
• Clear this site and reload every tab in this window — Alt+Shift+W

You can also right-click the toolbar icon to reopen the current page in a fresh incognito window after a deep clear.

Privacy: ClearCache collects nothing, sends nothing to any server, and stores nothing. It only clears data — the opposite of tracking. Every line of code is open source on GitHub.

Available in 20 languages including English, Spanish, French, German, Italian, Dutch, Polish, Portuguese (BR), Russian, Turkish, Japanese, Korean, Chinese (Simplified and Traditional), Vietnamese, Indonesian, Thai, Hebrew, Arabic, and Persian. Free and open source under GPL-3.0.
```

**Description** (descripción de la página de detalles):

¿Actualizaste un sitio web pero sigues viendo la versión vieja? ClearCache lo soluciona con un solo clic.

También puedes hacer clic derecho en el icono para reabrir la página actual en una ventana de incógnito nueva tras una limpieza profunda.

Privacidad: ClearCache no recopila nada, no envía nada a ningún servidor y no almacena nada. Solo borra datos — lo contrario de rastrear. Todo el código es abierto en GitHub.

Disponible en inglés y español. Gratis y de código abierto bajo GPL-3.0.
